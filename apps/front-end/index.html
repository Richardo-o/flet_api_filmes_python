🎬 Catálogo de Filmes

Sistema completo desenvolvido com Flask + Flet + Tailwind CSS. A aplicação permite listar e cadastrar filmes consumindo endpoints REST documentados com Swagger.

Sobre o Projeto Como Executar

🚀 Tecnologias Utilizadas

🐍

Flask Backend

API REST construída com Flask utilizando Blueprints, Swagger e validação de dados com Pydantic.

🖥️

Flet Frontend

Interface desktop moderna em Python utilizando Flet consumindo os endpoints da API.

🎨

Tailwind CSS

Landing page estilizada utilizando Tailwind CSS com layout responsivo e moderno.

📋 Funcionalidades

✔ Listagem de Filmes

Os filmes cadastrados são consumidos através de endpoints GET da API Flask.

✔ Cadastro de Filmes

Formulário integrado ao endpoint POST utilizando validação com Pydantic.

⚙️ Como Executar 

# Clonar repositório

git clone https://github.com/Richardo-o/flet_api_filmes_python.git

# Instalar dependências

pip install flask
pip install flask-restx
pip install flask-cors
pip install pydantic
pip install flet
pip install requests

# Executar backend

python app.py

# Executar frontend

python main.py

Projeto desenvolvido para atividade acadêmica utilizando Flask, Flet e Tailwind CSS.
